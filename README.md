# ☕ IGRUS 랜덤 커피챗 매칭기 (Coffee Chat Matcher)

> **인하대학교 IGRUS 동아리원들을 위한 스마트 랜덤 조 편성 웹 어플리케이션**

## 📖 프로젝트 소개
동아리 내 네트워킹과 친목 도모를 위한 **'커피챗(Coffee Chat)' 조 편성**을 쉽고 재미있게 만들어주는 단일 페이지 웹 어플리케이션(SPA)입니다. 기존의 단순 랜덤 매칭 방식의 단점을 보완하여, **멘토와 멘티가 한 조에 치우치지 않고 골고루 섞일 수 있도록 스마트 분배 알고리즘을 적용**했습니다.

## ✨ 주요 기능 및 특징

* 🎨 **IGRUS 오피셜 테마 적용**
  * IGRUS 공식 웹사이트의 아이덴티티를 살린 **화이트 & 민트 그린(Mint Green)** 테마를 적용하여 시각적인 통일성과 산뜻함을 주었습니다.
* ⚖️ **스마트 라운드 로빈(Round-Robin) 알고리즘**
  * '멘토 명단'과 '멘티 명단'을 따로 입력받습니다.
  * 멘토를 먼저 각 조에 1명씩 우선 배치한 뒤, 남은 자리에 멘티를 무작위로 분배하여 **모든 조가 균형 잡힌 인원 구성**을 갖추도록 설계되었습니다.
* ✨ **인터랙티브 3D 애니메이션**
  * 매칭 완료 시 밋밋하게 결과가 뜨는 대신, 순서대로 카드가 촤르륵 뒤집히는 **물리 기반(Cubic-Bezier) 3D 카드 애니메이션**을 추가하여 조 추첨의 기대감과 재미를 극대화했습니다.
* ⚡ **Zero-Dependency (무설치 구동)**
  * React, Vue 등의 무거운 프레임워크나 외부 라이브러리 설치(`npm install`)가 전혀 필요 없습니다.
  * 순수 HTML, CSS, Vanilla JavaScript만으로 작성되어 **`index.html` 파일 하나만 있으면 어떤 PC, 스마트폰 브라우저에서도 즉시 실행**됩니다.

## 🚀 실행 및 사용 방법

본 프로젝트는 별도의 서버나 컴파일 과정이 필요하지 않습니다.

### 1. 실행 방법
1. 이 레포지토리의 코드를 다운로드하거나 클론합니다.
2. 폴더 내에 있는 **`index.html`** 파일을 더블클릭하여 엽니다. (Chrome, Edge, Safari, Whale 등 모든 웹 브라우저 지원)

### 2. 사용 가이드
1. **멘토 명단 입력:** 조의 리더 역할을 할 멘토의 이름을 입력합니다. (쉼표 `,` 또는 띄어쓰기로 구분)
2. **멘티 명단 입력:** 매칭에 참여할 멘티들의 이름을 입력합니다.
3. **목표 조당 인원 설정:** 한 조에 몇 명이 배정될지 목표 인원을 선택합니다. (2명 ~ 5명)
4. **[매칭 시작 🎲] 버튼 클릭:** 카드가 뒤집히며 랜덤으로 편성된 조 결과를 확인합니다!

> **💡 참고 사항:** 전체 인원이 '목표 조당 인원'으로 딱 떨어지지 않는 경우, 알고리즘이 자동으로 남는 인원을 각 조에 균등하게 분배합니다.
