# ☕ IGRUS 선후배 랜덤 커피챗 매칭기 (Coffee Chat Matcher)

> **인하대학교 IGRUS 동아리원들을 위한 균등 분배형 랜덤 조 편성 웹 어플리케이션**

## 📖 프로젝트 소개
동아리 내 네트워킹과 친목 도모를 위한 **'선후배 커피챗(Coffee Chat)' 조 편성**을 지원하는 웹 툴입니다. 기존의 완전 무작위 매칭이 가진 단점(특정 조에 선배나 후배만 몰리는 현상)을 해결하기 위해, **선배(🌳)와 후배(🌱) 입력을 분리하여 조마다 균형 있게 섞이도록 설계**했습니다.

## ✨ 주요 기능 및 특징

* 🎨 **IGRUS 오피셜 라이트 테마**
  * IGRUS 공식 웹사이트의 정체성을 반영한 **화이트 & 민트 그린(Mint Green)** 컬러 스키마를 사용하여 산뜻하고 트렌디한 느낌을 줍니다.
* ⚖️ **균등 우선 분배 알고리즘 (Round-Robin)**
  * 대화를 매끄럽게 이끌어줄 **'선배(🌳)'**들을 생성된 조에 1명씩 우선적으로 균등 배치합니다.
  * 이후 남은 자리에 **'후배(🌱)'**들을 무작위로 배정하여 모든 조가 치우침 없는 인원 구성을 갖추게 합니다.
* 🎲 **인터랙티브 3D 카드 뒤집기 애니메이션**
  * 결과 도출 시 물리 기반 큐빅 베지에(Cubic-Bezier) 곡선을 적용한 카드 플립 효과를 주어 조 추첨의 시각적 재미를 더했습니다.
* ⚡ **Zero-Dependency (순수 웹 스택)**
  * 외부 라이브러리나 서버 구동 환경이 필요 없는 **Single Page Application**입니다.
  * `index.html` 단일 파일 형태로 제공되어 모바일, 태블릿, PC 등 어떤 브라우저에서도 즉시 실행 가능합니다.

---

## 🚀 사용 가이드

1. **`index.html`** 파일을 웹 브라우저로 실행합니다.
2. **선배 명단 입력:** 조를 이끌어 줄 선배 부원들의 이름을 입력합니다. (쉼표 `,` 나 띄어쓰기로 구분)
3. **후배 명단 입력:** 커피챗에 참여할 후배 부원들의 이름을 입력합니다.
4. **목표 조당 인원 설정:** 한 테이블에 배정될 적정 타겟 인원을 선택합니다. (2명 ~ 5명)
5. **[커피 테이블 배치 🎲]** 버튼을 클릭하여 시차 애니메이션과 함께 무작위로 매칭된 결과를 확인합니다.
